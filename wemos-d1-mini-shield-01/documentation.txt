ulno-shield 1

assumption lcd screen up

d3:  0 flash and left button
d6: 12 right button
d5: 14 lower button
d8: 15 yellow led
d7: 13 red led
d4:  2 on boad led (reversed low: on, high: off)
d1:  5 scl (lcd), lcd is upside down
d2:  4 sda (lcd)

Example for buttons and leds:
from machine import Pin

d7=Pin(13,Pin.OUT)
d7.high()
d7.low()

d5=Pin(14,Pin.IN,Pin.PULL_UP)
d5.value()

For lcd screen:
import ssd1306
from machine import I2C, Pin
i2c = I2C(sda=Pin(4), scl=Pin(5))
display = ssd1306.SSD1306_I2C(128,64,i2c)
display.fill(1)
display.show()

display.text("awesome",24,0)
for y in range(56): display.scroll(0,1); display.show()
for y in range(56): display.scroll(0,-1); display.show()

import network
